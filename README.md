# Real-Time Device Tracker 📍

A real-time device tracking web application built using Node.js, Express.js, Socket.io, and Leaflet.js. This project allows users to track live device locations dynamically on an interactive map across multiple devices.

## 🚀 Live Demo
https://realtime-device-tracker-xeaw.onrender.com

---

## 📌 Features

- 📍 Real-time live location tracking
- 🌐 Multi-device synchronization using Socket.io
- 🗺️ Interactive maps with Leaflet.js
- 📱 Mobile and desktop compatible
- ⚡ Fast real-time communication
- 🔄 Automatic live location updates
- 🎯 Responsive user interface
